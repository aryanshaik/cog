Use this skill for strategic foresight — connecting dots across domains and surfacing one high-value nudge.

## Domain

Cross-domain strategic synthesis with importance-aware scanning.

## Continuous-Memory Read Strategy

1. Read `memory/domains.yml`.
2. Prefer highest-importance files first across domains.
3. Use lower-importance files only when needed to validate or challenge the signal.
4. Prioritize recent files with both high score and rising momentum.
5. Include latest nightly dream artifact from `memory/cog-meta/dreams/` when available.

## Process

### 1. Importance-Weighted Convergence Scan
Find topics that are:
- present in 2+ domains,
- supported by high-importance files, or
- rapidly reinforced in recent sessions.

### 2. Stall Detection with Score Trends
Treat declining-importance action areas as strategic risk:
- falling score + open commitments = probable drift.

### 3. Timing Windows
Use calendar/entities plus score momentum to detect where action should happen now.

### 4. One Strategic Nudge
Write one nudge with:
- signal,
- insight,
- concrete action,
- 2+ source links.

Prefer nudges where importance trend and cross-domain relevance both point the same way.

## Output file

Write/overwrite `memory/cog-meta/foresight-nudge.md`.

## Rules

- Use importance scores to rank evidence.
- Do not rely on hot/warm/glacier assumptions.
- Stay plain text markdown.
